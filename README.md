# 4Hunnid Shopify Theme

---

## Use

##### Upload the current theme to a store for dev

```
$ yarn dev
```

##### Retrieve theme files from Shopify

```
$ yarn pull
```

##### Retrieve theme files from development store

```
$ yarn pull-dev
```

##### Upload local theme files to Shopify

```
$ yarn push
```

##### Log out of current Shopify account

```
$ yarn logout
```
